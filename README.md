# `typer-chucknorris`

**Usage**:

```console
$ typer-chucknorris [OPTIONS] COMMAND [ARGS]...
```

**Options**:

* `--install-completion`: Install completion for the current shell.
* `--show-completion`: Show completion for the current shell, to copy it or customize the installation.
* `--help`: Show this message and exit.

**Commands**:

* `chuck`: Get a random bad a$$ Chuck Norris quote...

## `typer-chucknorris chuck`

Get a random bad a$$ Chuck Norris quote from the CLI!!!

**Usage**:

```console
$ typer-chucknorris chuck [OPTIONS] [RANDOM]
```

**Arguments**:

* `[RANDOM]`: Get random quote  [default: True]

**Options**:

* `-c, --categories`: Retrieve a list of available categories.
* `-cs, --category-select TEXT`: Select category and retrieve a quote from the entered category
* `-s, --search TEXT`: Free text search to find relavent joke
* `-v, --version`
* `--help`: Show this message and exit.
